# blqw.Serialization
任意对象的序列化和反序列化


####再也不需要受限于系统的``[Serializable]``特性  
* 理论上将任何实例对象都序列化和反序列化  
* 适用场景一,分布式数据存储,如:Cache,Session,各种分布式数据库在不同服务器间的数据传递  
* 适用场景二,服务器日志将数据序列化后存入日志仓库,反序列化后不会丢失任何内容  

## 欢迎测试,如果有无法序列化或反序列化的类型,欢迎提交BUG
